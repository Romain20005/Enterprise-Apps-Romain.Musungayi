# Enterprise Apps Romain.Musungayi

## Beschrijving

Dit project is een prototype van een website voor een Anderlechtse NGO die zich inzet voor gemeenschapsopbouw en hulp aan mensen in een kwetsbare situatie.

De applicatie werd ontwikkeld met Spring Boot, Thymeleaf, H2 Database en Tailwind CSS.

## Functionaliteiten

* Overzicht van evenementen
* Toevoegen van nieuwe evenementen
* Validatie van invoer
* Detailpagina per evenement
* About pagina
* Contact pagina
* Navigatiemenu op elke pagina
* Footer op elke pagina

## Gebruikte technologieën

* Java
* Spring Boot
* Spring MVC
* Spring Data JPA
* Thymeleaf
* H2 Database
* Tailwind CSS
